# gaia-fauna-contornos

Contornos SDM (GeoJSON) do projeto Gaia-fauna / FieldNote, servidos via **jsDelivr CDN**.

Cada arquivo em modelos/<slug>.geojson e o contorno do modelo de distribuicao (SDM)
de uma especie (camadas: consenso, maxent, rf, lgbm, incerteza).

Servidos em:
`
https://cdn.jsdelivr.net/gh/SetegCE/gaia-fauna-contornos@main/modelos/<slug>.geojson
`

Gerados em lote pelo treino do catalogo (Gaia-fauna). Read-only; atualizados por lote.
